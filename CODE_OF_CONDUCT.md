# Code of Conduct

This project follows a simple community code of conduct to keep contributions friendly and productive.

## Our Standards
- Be respectful and professional.
- Assume good intent and give constructive feedback.
- Avoid harassment, hate speech, and discriminatory behavior.
- Keep discussions focused on code and project goals.

## Reporting Issues
If you face or notice inappropriate behavior, open an issue or contact the repository owner privately. Confidentiality will be respected.

## Enforcement
Project maintainers may remove comments, reject contributions, or block users who violate this code of conduct. The aim is to maintain a welcoming environment for all.
